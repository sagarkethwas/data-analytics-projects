# Data Analytics & Machine Learning Projects

This repository contains data analysis and machine learning projects developed using Python, SQL and Power BI.

## Projects Included

### 1. Crop Prediction System
- Machine learning based crop recommendation system
- Inputs: soil nutrients, rainfall, temperature and humidity
- Tools: Python, Pandas, Scikit-learn

### 2. Sales Dashboard (Power BI)
- Interactive sales dashboard for business insights
- KPIs: Total Sales, Profit, Region-wise and Monthly trends
- Tool: Power BI

### 3. Fake News Detection System
- NLP based fake news classification
- Techniques: TF-IDF and Logistic Regression
- Tool: Python
###4. Exploratory Data Analysis (EDA)
Performed EDA on a Telecom Customer Churn dataset.
Analyzed churn rate, tenure, contract type, internet service and customer demographics.
Extracted business insights and retention strategies.
Tools: Python, Pandas, Matplotlib, Seaborn
